User List Application - test task for adverster

Start:
git clone https://github.com/Iamvagabund/adverster.git

cd adverster,
npm install

Для запуску проекту в режимі розробки використовуйте команду:
npm run dev


Щоб створити продакшн-версію:
npm run build
