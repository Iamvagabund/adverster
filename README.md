User List Application - test task for adverster

Start:
git clone https://github.com/Iamvagabund/adverstertest.git

cd adverstertest
npm install

Для запуску проекту в режимі розробки використовуйте команду:
npm run dev


Щоб створити продакшн-версію:
npm run build


Технології:
JavaScript
Webpack
CSS
HTML
